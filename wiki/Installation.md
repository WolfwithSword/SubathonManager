# Installation

[Download the latest](https://github.com/WolfwithSword/SubathonManager/releases/latest) 
zip file and extract to a location of your choosing. 

# Updating 

To update manually from a previous version, we recommend backing up your old installation, then overwriting with the latest.

To update automatically, go to the Settings tab and check for updates. If one is available, you will be prompted to update in place.

All your settings and configuration should be preserved after updating.
